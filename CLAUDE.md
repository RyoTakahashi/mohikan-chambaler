# Mohikan-chanbaler

## 概要
Mohikan-chanbalerは、ヘアスタイルの一つであるモヒカンを使って相手と戦う戦略型アクションゲームです。
このゲームは大きく２つのパートで成り立っており、
1. モヒカンの育成
2. モヒカンを使った戦い
となっています。モヒカンにはいくつか種類が存在し、土台となる頭の形と、モヒカンの形状を組み合わせることで、様々なモヒカンが生まれます。また、モヒカンには攻撃力、防御力、スタミナ、HPのステータスが存在し、これらのステータスを育成することで、モヒカンを強化することができます。
更にモヒカンの完成度を高めるためのアイテム（整髪剤やカラーリング剤など）も存在し、これらを使うことでモヒカンの性能を更に引き出すことができます。
対戦には3体のモヒカンを用意して、団体戦方式で戦いをします。そのため、先に2勝した方が勝利となります。
戦闘に勝利するとモヒカンの育成や強化に役立つアイテムやモヒカンの土台をゲットすることができます。

## ゲームの背景世界
199X年、日本はモヒカンブームの真っ只中にあった。街にはモヒカン刈りの若者が溢れ、モヒカンを愛でるためのコンテストが各地で開催されていた。そんな中、突如として現れた謎の組織「モヒカン教団」が、日本中のモヒカンを奪い去ろうと企む。プレイヤーは、モヒカンを愛する一人の若者として、モヒカン教団からモヒカンを取り戻すために立ち上がる。プレイヤーはモヒカンを育成し、モヒカン教団と戦い、日本中のモヒカンを取り戻すことができるのか？

## ゲームのシステム

### ゲームの流れ
まずゲームのシナリオを進めていくと、対戦が決まります。対戦が決まるとまず、対戦に出場させる3体のモヒカンを選びます。その後、5ターンの間モヒカンを育成します。5ターン経過すると、対戦が始まります。対戦に勝利すると、モヒカンの育成や強化に役立つアイテムやモヒカンの土台をゲットすることができます。対戦のあとは、またシナリオに戻り、育成と対戦を繰り返します。シナリオを進めていくと、より強いモヒカン教団のメンバーと戦うことになります。最終的にはモヒカン教団のボスと戦い、勝利するとゲームクリアとなります。

### 対戦決定後のモヒカン選択
対戦が決まると、まず対戦に出場させる3体のモヒカンを選びます。モヒカンは土台が10種類、モヒカンの形状が10種類で全部で100種類あり、それぞれのモヒカンには攻撃力、防御力、スタミナ、HPの4種類のステータスがあります。モヒカンの基本ステータスは、モヒカンの形状と頭の形の組み合わせによって決まりますが、その後の育成によって強化をすることができます。

### 対戦までのモヒカン育成
対戦が決まり、出場するモヒカンを決めたあとは、全部で5ターンの間モヒカンを育成することができます。育成ポイントを消費して、モヒカンのステータスを強化することができます。育成の行動パターンは大きく下記の6種類です。
1. シャンプー ... 髪の質を上げることで攻撃力が中程度に上昇し、防御力とスタミナも少し上昇します
2. トリートメント ... ツヤツヤの髪となり毛根もケアされることでスタミナが大幅に上昇し、攻撃力と防御力も少し上昇します
3. ストレートパーマ ... 髪がまっすぐになり、攻撃力がとんでもなく上昇しますが、防御力とスタミナが少し下がります
4. ブラシ ... 髪が整い、防御力が中程度に上昇し、攻撃力とスタミナも少し上昇します
5. ヘアオイル ... 髪にツヤが出て、防御力がとんでもなく上昇しますが、攻撃力とスタミナが少し下がります
6. アイテム利用 ... 整髪剤やカラー剤など、アイテムを使用することでステータスを上昇させることができます
5ターン経過すると、対戦が始まります。各ターンで育成できるモヒカンは1体のみです

### モヒカンの戦闘
戦闘は1vs1で行われます。お互いのモヒカンが攻撃しあって、相手のHPを0にした方が勝利となります。ストリートファイターのような2D型対戦で、アクションパターンは下記の通りです。
1. 攻撃ボタン ... モヒカンを振り回し、相手のHPにダメージを与えます
2. 防御ボタン ... モヒカンを固くして、相手の攻撃を防ぎます
3. 必殺技ボタン ... それぞれのモヒカン特有の技を繰り出し、相手のHPに大ダメージを与えたり、スタミナを削ることができます
4. 右左移動 ... 移動を行い、相手との距離を詰めたり、離れたりします
対戦時の主要なゲージは下記です
- HPゲージ ... 0になると敗北となります
- 必殺技ゲージ ... 攻撃を当てることで溜まり、100%になると必殺技が使用可能になります
- スタミナゲージ ... 攻撃や防御をすることで消費し、ゲージが0のときは攻撃や防御ができなくなります。時間経過とともに回復します

### モヒカンのアイテム
モヒカンのアイテムは、モヒカンの育成や強化に役立つアイテムです。モヒカンのアイテムは、モヒカンの戦闘に勝利することでゲットすることができます。
- リーゼントの教科書 ... モヒカンにとってのバイブル。防御力とスタミナを著しく上昇させるが、攻撃力が大幅に減少する
- ケ◯プ ... 髪をガチガチに固めたいときに利用する。攻撃力が著しく上昇するが、防御力が大幅に減少する
- ギャッ◯ビーヘアワックス ... 髪を固めて形をかっこよくする。防御力とスタミナをやや上昇させる。
- 幻のアデランス ... 育成によりマイナスになっているパラメータを元々の値に戻す。
