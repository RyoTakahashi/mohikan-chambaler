# モヒカン・チャンバラ

スマートフォン向けWebブラウザゲーム。モヒカンを育て、モヒカン教団と戦う戦略型アクションゲーム。

## 遊び方

`index.html` をブラウザで開くだけで動作します。サーバー不要。

## ゲーム概要

**舞台:** 199X年の日本。謎の組織「モヒカン教団」が日本中のモヒカンを奪い去ろうと企んでいる。プレイヤーはモヒカンを育て、教団を倒すために戦う。

### ゲームの流れ

1. **モヒカン選択** — 対戦に出場させる3体のモヒカンを選ぶ
2. **育成フェーズ** — 5ターンの間、モヒカンを強化する
3. **団体戦** — 選んだ順番に従って1対1で戦う（先に2勝した方が勝利）
4. 勝利するとアイテムや新モヒカンを獲得し、次の相手へ

### 育成アクション

| アクション | 効果 |
|---|---|
| 🧴 シャンプー | ATK↑↑ DEF↑ STA↑ |
| ✨ トリートメント | STA↑↑↑ ATK↑ DEF↑ |
| 💈 ストパー | ATK↑↑↑↑ DEF↓ STA↓ |
| 🪮 ブラシ | DEF↑↑ ATK↑ STA↑ |
| 🫙 ヘアオイル | DEF↑↑↑↑ ATK↓ STA↓ |
| 🎒 アイテム使用 | 所持アイテムで強化 |

### 戦闘操作

| ボタン | 効果 |
|---|---|
| ◀ ▶ | 左右移動 |
| A（攻撃） | モヒカンを振り回して攻撃（スタミナ消費） |
| B（防御） | 相手の攻撃をガード（ダメージ大幅軽減） |
| S（必殺技） | ゲージが満タン時に使用可能。固有の大技を繰り出す |

### ゲージ

- **HPゲージ** — 0になると敗北
- **スタミナゲージ** — 攻撃・防御で消費、時間経過で回復。0では行動不能
- **必殺技ゲージ** — 攻撃を当てるごとに蓄積、100%で必殺技が使用可能

### モヒカンのステータス

頭の形（10種）× モヒカンの形（10種）= 最大100種類のモヒカン

| ステータス | 説明 |
|---|---|
| ATK | 攻撃力。相手に与えるダメージに影響 |
| DEF | 防御力。受けるダメージを軽減 |
| STA | スタミナ上限。行動できる回数に影響 |
| HP | 体力。0になると敗北 |

### アイテム

| アイテム | 効果 |
|---|---|
| 📗 リーゼントの教科書 | DEF・STA大幅↑、ATK↓↓ |
| 💨 ケープ | ATK大幅↑、DEF↓↓ |
| 🫙 ギャッツビーワックス | DEF・STA↑ |
| 💎 幻のアデランス | マイナスになったパラメータを基本値に戻す |

## 技術仕様

- **動作環境:** モダンブラウザ（Chrome / Safari / Firefox）
- **依存関係:** なし（外部ライブラリ不使用）
- **構成:** `index.html` 単一ファイル
- **描画:** Canvas API（バトル画面）
- **音声:** Web Audio API（効果音を動的生成）
- **対象デバイス:** スマートフォン（タッチ操作対応）、PC（マウス操作対応）
